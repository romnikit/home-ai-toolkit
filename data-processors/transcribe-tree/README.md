
# whisper-mps based media scanner.

## Needed tool to be installed.

    brew install pyenv ffmpeg

## Setup of the environment.

    1. Create the local folder to place there packages.

        cd ~
        mkdir -p ./whisper-mps-env
        cd ./whisper-mps-env

    2. Copy appropriate setup-whisper-env-*.sh script.

        cd ~/whisper-mps-env
        <... copy script ...>
        . ./setup-whisper-env-*.sh
